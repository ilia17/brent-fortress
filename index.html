Brent's Crypto Fortress

Defend Brent's Bitcoin from the US Army!

🌍
The Map
Enemies march along the brown path from the Army Camp (top-left) to Brent (bottom-right). Place towers on the green tiles to stop them.
⚡
Place Towers
Select a tower type from the bar at the bottom, then click any green tile to build it. You can't build on the path.
⬆️
Upgrade & Sell
Click a placed tower to upgrade it (costs crypto). Between waves the Shop opens — use it to sell towers you no longer need.
💰
Economy
You start with 200 ₿. Killing enemies earns crypto. Clearing a wave pays a wave bonus. If you run out, sell old towers.
💀
Enemy Types
Early waves send basic soldiers. Later waves mix in tanks, helicopters, and elite units. Every 3rd wave is a BOSS wave — be ready!
❤️
Lives
You have 20 lives. Each enemy that reaches Brent costs 1 life (bosses cost 5). Reach 0 and it's game over — Brent loses his Bitcoin!
